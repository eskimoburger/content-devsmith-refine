# Welcome to Lucky Draw Content Storybook

This Storybook showcases the interactive components used in the Lucky Draw application.

## Components

### LuckyCard
An interactive lucky draw card component featuring:
- Stacked card animation
- 3D tilt effects
- Card flipping animations
- Responsive design
- Prize reveal functionality

### TiltedCard
A reusable 3D tilted card component with:
- Mouse-tracking tilt effect
- Customizable scale and rotation
- Optional tooltip
- Overlay content support
- Mobile warning support

## Getting Started

Browse the components in the sidebar to see different variations and interactive demos.

## Technologies

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Motion (Framer Motion)
- Storybook 8

## Available Scripts

- `npm run storybook` - Start Storybook development server
- `npm run build-storybook` - Build Storybook for production
